# Cinnamon Indian Cuisine

A modern restaurant website for Cinnamon Indian Cuisine, showcasing authentic Indian dining experience.

## Getting Started

This is a Next.js project built with TypeScript and Tailwind CSS.

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

```bash
npm run build
```

### Deploy

```bash
npm run start
```

## Tech Stack

- Next.js 13+ (App Router)
- TypeScript
- Tailwind CSS
- React Icons
- Vercel Analytics

## Features

- Responsive design
- Modern UI/UX
- SEO optimized
- Fast performance
- Mobile-friendly
